# XO SpaceTech

Hackathon project — room dimension calculator and space planning tool built with Next.js. Helps estimate room layout and object placement for moves and spatial organization.

## Features

- Room dimension input and layout visualization
- Object placement planning UI
- Responsive Next.js front end

## Stack

Next.js 14 · React · Tailwind CSS · Framer Motion

## Run locally

```bash
npm install
npm run dev
```
